# Python Lab Practicals

This repository is a shared reference for all Python laboratory practicals performed during lab sessions.
It is intended to help everyone revise concepts, verify implementations, and stay aligned with lab progress.

---

## Repository Structure

```bash
.
├── Day01/
│   └── ...
├── Day02/
│   └── ...
├── Day03/
│   └── ...
└── README.md
```

Each DayXX directory corresponds to a single lab session and contains all practicals completed on that day.

---

## How to Use

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to a specific lab day:

```bash
cd Day01
```

Run a practical:

```bash
python evenOdd.py
```
